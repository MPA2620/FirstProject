This boilerplate is ready for HTML/CSS, Javascript, and JQuery projects. It includes a live server that lets you see HTML/JS/CSS changes with every save.

An Example JavaScript Hook.

Instructions:

To see live changes, please go to the root folder ('simple-boilerplate/') and run "npm start".

To start coding, start changing the files within src/css and src/js. Images can be stored in src/assets.
